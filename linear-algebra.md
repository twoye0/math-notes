# Linear Algebra

## 1. Vectors and Matrices
- 벡터와 행렬은 여러 변수 간 관계를
  구조적으로 표현하는 도구라고 이해하였다.

## 2. Covariance Matrix
- 공분산 행렬은 자산 간 수익률의
  변동성과 상관관계를 함께 나타낸다.
- 포트폴리오 위험 계산에 핵심적인 역할을 한다.

## 3. Portfolio Risk Interpretation
- 개별 자산의 분산뿐 아니라
  자산 간 상관관계가
  전체 포트폴리오 위험에 큰 영향을 준다는 점이 인상적이었다.

## 4. Connection to Finance
- CAPM 기반 포트폴리오 최적화 프로젝트에서
  공분산 행렬을 이용해
  포트폴리오 분산을 직접 계산해 보았다.
