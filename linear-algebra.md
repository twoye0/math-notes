# Linear Algebra

## 1. Vectors and Matrices
- 다변수 문제를 구조적으로 표현하는 도구로 이해하였다.

## 2. Matrix Operations
- 행렬 곱셈과 전치, 역행렬의 개념을 학습하였다.
- 계산 규칙보다는 구조적 의미에 집중하였다.

## 3. Eigenvalues and Eigenvectors
- 선형 변환의 성질을 이해하는 핵심 개념으로 학습하였다.
- 시스템의 안정성 해석과 연결될 수 있다고 느꼈다.

## 4. Covariance Matrix
- 자산 간 수익률의 변동성과 상관관계를 동시에 나타낸다.
- 포트폴리오 위험 계산의 핵심 도구로 활용하였다.

## 5. Application to Finance
- CAPM 기반 포트폴리오 최적화 프로젝트에서
  공분산 행렬을 이용해 포트폴리오 분산을 계산하였다.
